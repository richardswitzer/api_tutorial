What is an API? ü§î

An API (Application Programming Interface) is like a waiter at a restaurant. You (the customer) tell the waiter what you want, the waiter goes to the kitchen (the server), and brings back your food (the data)!

Today we're using the Cat as a Service API at https://cataas.com. It's a fun API that gives us random cat pictures! üéâ

2

Your First API Call üöÄ

The simplest way to call an API is through a URL (web address). Let's get a random cat picture!

// The basic endpoint for a random cat GET https://cataas.com/cat

Try it yourself! üëá

In web development, when we "call" an API, we're making an HTTP request. The most common type is a GET request, which asks for data.

Adding Parameters üé®

APIs often let you customize your request using parameters. Parameters are like toppings on a pizza - they modify what you get back!

// Add text to your cat image GET https://cataas.com/cat/says/Hello // Customize the size GET https://cataas.com/cat?width=500&height=300

Customize your cat! üé≠
